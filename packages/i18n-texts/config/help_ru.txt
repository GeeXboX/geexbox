ПОЛЕЗНЫЕ КОМАНДЫ
m  : Показать/Скрыть меню
o  : Показать таймер
s  : Включить TV-выход
Left : Отмена / Прокрутить назад
Right : Ok / Прокрутить вперед
Down : Прокрутить назад 1 minute
Up : Прокрутить вперед 1 minute
+/-  : Аудио-задержка +/- 100ms
*// : Громкость +/- 1

КОМАНДЫ ПРИ ВЫБОРЕ ФАЙЛОВ
d  : Установить каталог назначения
c  : Скопировать в каталог назначения
r  : Запрос на удаление файла
o  : Подтвердить удаление файла
p  : Воспроизвести весь каталог
l  : Рекурсивно смотреть весь каталог
u  : Добавить в плейлист
